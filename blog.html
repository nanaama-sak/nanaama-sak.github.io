Publish — GitHub Pages
  1. Create a free account at github.com.
  2. New repository → name it yourusername.github.io → Public.
  3. Click "uploading an existing file". Drag both files in. Rename blog.html → index.html.
  4. Export content.json below and include it in the same upload.
  5. Commit. Settings → Pages → source = main → Save.
  6. Visit https://yourusername.github.io — live in ~2 min.
  7. To update: export fresh content.json, paste into GitHub file editor, commit.

Images and CV are embedded as base64 in content.json. Resize large images before upload to keep file size manageable.
